# 路线图

## M0：工程与灰盒垂直切片

- 工程、拆分式主菜单/暂停菜单、双世界、第一人称、双手实体工具、连续防重叠摆放、组合砧板、允许犯错的传统工序、每日洗手、量酒器取水、有限重复补救、液体、客人限距提交、完成度评价、30 天近视进阶、两人宽封闭吧台、可通行柜体/抽屉、测试与资产管线。
- 完成标准：在 Godot 4.7.1 .NET 中可从接单前自由操作一路游玩到第 30 天周目结束；菜单鼠标/键盘状态正确，柜体开启不困住玩家，正确流程、错误工具/材料、比例偏差、缺水、未洗手和手动清废均有可感知且可恢复的结果，无阻断错误。
